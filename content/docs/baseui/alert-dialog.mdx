## Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `actionsRef?` | `React.RefObject<AlertDialogRoot.Actions \| null> \| undefined` | `-` | A ref to imperative actions. - `unmount`: When specified, the dialog will not be unmounted when closed. Instead, the `unmount` function must be called to unmount the dialog manually. Useful when the dialog's animation is controlled by an external library. - `close`: Closes the dialog imperatively when called. |
| `handle?` | `DialogHandle<Payload> \| undefined` | `-` | A handle to associate the alert dialog with a trigger. If specified, allows external triggers to control the alert dialog's open state. Can be created with the AlertDialog.createHandle() method. |
