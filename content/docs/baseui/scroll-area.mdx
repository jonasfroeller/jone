## Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `xStart` | `number` | `-` | - |
| `xEnd` | `number` | `-` | - |
| `yStart` | `number` | `-` | - |
| `yEnd` | `number` | `-` | - |
| `orientation?` | `'vertical' \| 'horizontal' \| undefined` | `vertical` | Whether the scrollbar controls vertical or horizontal scroll. |
| `keepMounted?` | `boolean \| undefined` | `false` | Whether to keep the HTML element in the DOM when the viewport isn’t scrollable. |
