## Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `keepMounted?` | `boolean \| undefined` | `false` | Whether to keep the HTML element in the DOM when the radio button is inactive. |
| `value` | `Value` | `-` | The unique identifying value of the radio in a group. |
| `disabled?` | `boolean \| undefined` | `-` | Whether the component should ignore user interaction. |
| `required?` | `boolean \| undefined` | `-` | Whether the user must choose a value before submitting a form. |
| `readOnly?` | `boolean \| undefined` | `-` | Whether the user should be unable to select the radio button. |
| `inputRef?` | `React.Ref<HTMLInputElement> \| undefined` | `-` | A ref to access the hidden input element. |
