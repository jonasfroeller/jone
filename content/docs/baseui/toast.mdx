## Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `anchor?` | `Element \| null \| undefined` | `-` | An element to position the toast against. |
| `side?` | `Side \| undefined` | `top` | Which side of the anchor element to align the toast against. May automatically change to avoid collisions. |
| `toast` | `ToastObject<any>` | `-` | The toast object associated with the positioner. |
| `children?` | `React.ReactNode` | `-` | - |
| `timeout?` | `number \| undefined` | `5000` | The default amount of time (in ms) before a toast is auto dismissed. A value of `0` will prevent the toast from being dismissed automatically. |
| `limit?` | `number \| undefined` | `3` | The maximum number of toasts that can be displayed at once. When the limit is reached, the oldest toast will be removed to make room for the new one. |
| `toastManager?` | `ToastManager \| undefined` | `-` | A global manager for toasts to use outside of a React component. |
