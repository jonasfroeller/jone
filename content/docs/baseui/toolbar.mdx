## Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `disabled?` | `boolean \| undefined` | `false` | When `true` the item is disabled. |
| `focusableWhenDisabled?` | `boolean \| undefined` | `true` | When `true` the item remains focuseable when disabled. |
| `defaultValue?` | `React.ComponentProps<'input'>['defaultValue'] \| undefined` | `-` | - |
| `orientation?` | `ToolbarRoot.Orientation \| undefined` | `horizontal` | The orientation of the toolbar. |
| `loopFocus?` | `boolean \| undefined` | `true` | If `true`, using keyboard navigation will wrap focus to the other end of the toolbar once the end is reached. |
